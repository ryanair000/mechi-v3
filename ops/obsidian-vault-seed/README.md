# Mechi Obsidian Vault Seed

This folder seeds the dedicated Mechi Obsidian vault used by the OpenClaw `control` agent on the host.

It is intentionally small and operational:

- `Ops/` for durable COO notes and runbooks
- `Product/` for product state summaries
- `Infra/` for host and deployment context
- `People/` for operator notes

These notes are internal memory, not the source of truth for live production state.
